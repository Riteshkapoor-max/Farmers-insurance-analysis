# 🌾 Farmers Insurance Analysis using SQL

> A comprehensive SQL data analysis project on agricultural insurance data to uncover insights into farmer coverage, premium contributions, insured land area, and government support across Indian states and districts.

## 📌 Project Overview

The **Farmers Insurance Analysis** project focuses on analyzing crop insurance data using **MySQL**. The objective is to answer real-world business questions related to insurance coverage, premium distribution, farmer participation, and regional agricultural trends.

The analysis demonstrates strong SQL skills including data retrieval, aggregation, joins, subqueries, Common Table Expressions (CTEs), window functions, and data manipulation operations. The project is based on the **FarmersInsuranceData** dataset, which contains detailed information about agricultural insurance across India. :contentReference[oaicite:0]{index=0}

## 🎯 Business Objectives

The project aims to answer questions such as:

- Identify states with maximum farmer coverage
- Analyze premium contributions by farmers, state governments, and central government
- Compare insured land area across districts
- Find high-performing and low-performing states
- Calculate insurance coverage ratios
- Analyze Sum Insured across different regions
- Identify trends using ranking and window functions
- Generate insights for better decision-making
